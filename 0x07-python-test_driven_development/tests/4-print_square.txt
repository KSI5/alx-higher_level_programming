# 4-print_square.txt

=============================
How to Use 4-print_square.py
=============================

This module defines a function ``print_square(size)`` that prints a square with the character '#'.

Usage
=====

The function takes an integer ``size`` as input and prints a square of side length ``size`` using the character '#'.

::

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

::

    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########

::

    >>> print_square(0)
    (no output)

::

    >>> print_square(1)
    #

Invalid Inputs
==============

If the input ``size`` is not an integer, a ``TypeError`` exception is raised with the message "size must be an integer".

::

    >>> print_square(4.5)
    Traceback (most recent call last):
        ...
    TypeError: size must be an integer

If the input ``size`` is less than 0, a ``ValueError`` exception is raised with the message "size must be >= 0".

::

    >>> print_square(-3)
    Traceback (most recent call last):
        ...
    ValueError: size must be >= 0
